Privacy Policy for Bread Converter

Effective Date: 09-02-25

Introduction

Bread Converter is designed to provide a seamless currency conversion experience. We respect your privacy and ensure that no personal data is collected, stored, or shared.

Information We Collect

This extension does not collect, track, or store any personal information.

The only data stored locally is your default currency preference, which is saved using Chrome’s storage API for convenience.

Currency conversion rates are retrieved from the HexaRate API but are not linked to any user-identifiable data.

Permissions We Use and Why

  • activeTab: Allows the extension to access the current tab only when the user interacts with it, ensuring privacy and security.
  • scripting: Enables the extension to inject a script that detects selected currency values and displays a conversion popup, without modifying webpage content.
  • storage: Used only to remember the user's default currency setting within Chrome's local storage.

How We Use This Information

The extension functions entirely on the user’s device and does not transmit any data externally.

No analytics, tracking, or background data collection occurs.

Third-Party Services

We use the HexaRate API to fetch live currency rates.

No personally identifiable information is sent to or stored by this service.

Security & User Control

The extension runs only when manually triggered by the user, ensuring privacy.

Users can remove the extension at any time from their browser settings.
